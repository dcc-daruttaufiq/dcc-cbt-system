# Server

Folder ini disiapkan untuk backend/API dari aplikasi DCC CBT (belum diimplementasikan).

Rencana peruntukan:
- REST/GraphQL API untuk autentikasi, bank soal, sesi ujian, dan penilaian
- Koneksi ke database
- Middleware autentikasi (dikonsumsi oleh `client/src/services/api.js`)

Stack backend akan ditentukan pada tahap pengembangan berikutnya.
